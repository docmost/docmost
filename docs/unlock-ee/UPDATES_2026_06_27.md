## OIDC Implementation Note

### Fixed Callback URL
```
POST /api/sso/oidc/callback
```

**Advantages**:
- No dynamic path generation
- No UUID per instance
- Consistent across all deployments
- Simpler OIDC provider configuration
- Easier to document and support

**Provider Configuration**:
```
Redirect URI: https://your-domain.com/api/sso/oidc/callback
State Parameter: Required for CSRF protection
Code Challenge: PKCE recommended
```
